\doxysubsection{Detailed Description}
This class is used by the \mbox{\hyperlink{class_o_s_m_data_parser}{O\+S\+M\+Data\+Parser}} class to draw the ways. It holds references not only to the ways but also the nodes referred to by the ways.

It uses the \mbox{\hyperlink{class_lat_lon_utility}{Lat\+Lon\+Utility}} class to take the latitude and longitude in the O\+SM map and the \mbox{\hyperlink{class_screen_utility}{Screen\+Utility}} class to figure out where on the Cairo surface the the particular point on the particular way goes.
