\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_capstone_mapping}{Capstone\+Mapping}} class is used by the \mbox{\hyperlink{class_capstone_drawing_area}{Capstone\+Drawing\+Area}} class to do work. The create\+Big\+Map method spawns new thread to initiate downloading from the Open Street Maps mirror as well as the O\+SM expat X\+ML parser.

This class also holds smart pointers, mostly shared pointers, to the other workers and Utility classes.
