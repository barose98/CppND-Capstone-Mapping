\doxysubsection{Detailed Description}
This class has some private variables and public methods to help the main G\+TK application to keep track of the offset, which specifies where the Cairo Drawling area\textquotesingle{}s source is compared to the big map In the \char`\"{}background\char`\"{}.

The \char`\"{}\+Calculate-\/\char`\"{} methods also allow for figuring screen points from a percentage and vice versa.
