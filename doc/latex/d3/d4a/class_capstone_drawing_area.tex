\doxysubsection{Detailed Description}
This class subclasses Gtk\+Drawling\+Area and overrides three event handlers. Additionally, it has an initialize method that starts the whole downloading and drawing process. It also calls the timed\+Redraw method while the \mbox{\hyperlink{class_cairo_drawer}{Cairo\+Drawer}} is drawing, as this turns out to be a resource intensive process and the user needs to be shown that the program is chewing on the drawing process and is not stuck.
